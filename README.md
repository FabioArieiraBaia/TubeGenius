## üöÄ Features

### 1. High-CTR Thumbnail Generation
Leveraging the **Gemini 2.5 Flash Image** model, TubeGenius generates 3 unique thumbnail variations simultaneously based on a single video title.
*   **Psychological Triggers:** Automatically injects visual cues for "Shock/Reaction", "Storytelling/Conflict", and "Mystery/Curiosity".
*   **Parallel Processing:** Uses asynchronous parallel API calls to deliver multiple options instantly.
*   **Reference Face Upload:** Supports "Image-to-Image" generation to keep your character/face consistent across thumbnails.

### 2. Viral Title Generator (Tabajara Flow)
Uses a specialized Data Science persona prompt to analyze video topics.
*   **Viral Scoring:** Assigns a 0-100% probability score to each title.
*   **Cognitive Bias Tags:** Explains the science behind the title (e.g., "Loss Aversion", "FOMO", "Negativity Bias").
*   **Direct Workflow:** One-click conversion from a generated title to a thumbnail creation workflow.

### 3. Channel Banner Studio
Creates professional 16:9 branding assets for channel headers.
*   **Branding focus:** Optimizes for text clarity and safe-zone centering.
*   **Style Variations:** Automatically creates Minimalist, Illustrative, and Cinematic options.
